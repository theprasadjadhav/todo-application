# todo-application
## Run Server Side
change directory from root folder to server folder <br/>
`cd server`<br/>
install necessary packages <br/>
`npm install` <br/>
run the server <br/>
`node App.js` <br/>
<br/>
## Run Client Side 
change directory from root folder to client folder <br/>
`cd client` <br/>
install necessary packages <br/>
`npm install` <br/>
run the application (client side) <br/>
`npm start`
